For setting up this project in your device you should download the zip file or clone the repository.
open the dsc folder in vscode or any other code editor.
open terminal (in vscode or the other code editor)
run the command npm install to install all the required dependencies
once its done run nodemon.cmd or nodemon server.js
it will take same time. As soon as the database is connected open Chrome
open localhost:3400/signup.
